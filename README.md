# from video to audio
программа для сохранения аудио дорожки из видео файла

### Технологии:
- Python 3.7
- moviepy

### Запуск:
установить и запустить виртуальное окружение:
```
python3 -m venv venv
source venv/bin/activate
```

установить необходимую библиотеку:
```
pip install --upgrade pip
pip install moviepy
```
